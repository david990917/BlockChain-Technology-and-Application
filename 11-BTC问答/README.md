# 11-BTC问答

1. 转账交易的时候，是收款方不在线怎么办？

   收款方不必连在比特币网络中

2. 有没有可能收款者的账户是全节点从来没听说过的？

   可能。创建的时候只有自己知道，只有第一次收款的时候，其他节点才知道。

3. 私钥丢失了怎么办？

   没有办法，钱变成了死钱，永远取不出来了。

   最著名的一个例子就是 **Mt. Gox**  

4. 私钥泄露了怎么办？

   进款转移账户上面钱到另外的账户上。

5. 如果转账的时候写错了地址怎么办？

   没有办法取消已经发布的交易。

6. 矿工偷 nounce

   有没有可能偷 别人挖出来的 nounce，需要改 coinbase 区块里面收款人地址，然后就改变了根哈希值。

7. 交易费可以看作是交易的小费，实现怎么知道给谁？
